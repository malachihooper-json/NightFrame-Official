# Drone Android

## Mobile Mesh Scout

.NET MAUI implementation of NIGHTFRAME node for mobile deployment and RF environment scanning.

## Features

- RF Scanning: Continuous Wi-Fi/Bluetooth monitoring for mesh discovery
- GPS Integration: High-accuracy telemetry for RF fingerprinting training data
- Mobile Inference: On-device AI accelerators (NPU) via ONNX where supported
- Mesh Connectivity: Connects to Orchestrator as lightweight scout node

## Deployment

Target API Level: 33+ (Android 13.0 or higher)
Permissions required: Location (Fine), Bluetooth (Admin), Wi-Fi State
