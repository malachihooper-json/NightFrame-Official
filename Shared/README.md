# Shared Components

## Interface Definitions and Message Contracts

Contains core abstractions and data models used by all nodes in the NIGHTFRAME mesh network.

## Principal Interfaces

- INeuralCompute: Protocol for sharding and executing model inference.
- ICellularIntelligence: Interface for RF sensing and location prediction.
- INetworkNode: Base definition for node discovery and identity management.

## Data Contracts

Includes Protobuf definitions and C# message classes for gRPC and SignalR communication, ensuring type safety and binary compatibility across the mesh.
